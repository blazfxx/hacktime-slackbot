Commands

/3t

Tic Tac Toe

Start a game with /3t start, make moves with /3t move 1–9. Beat the bot if you can.

/rps

Rock Paper Scissors

Quick one-shot game. Try /rps rock, /rps paper, or /rps scissors.

/flip

Coin Flip

Can't decide? Let fate choose. Instant heads or tails.

/hello

Help

Lists all available commands and how to use them. Great for first-timers.

How it works

01

Add to your workspace

Click the button above and authorize the bot in your Slack workspace.

02

Invite to a channel

Type /invite @minigames in any channel.

03

Start playing

Run any slash command. Responses are only visible to you so you won't spam the channel.
